What Kheyr does

Modern SMS inbox

Thread list with avatars, pinned conversations, search, archive, and a Telegram-inspired layout without copying third-party branding.

On-device spam filtering

Global spam rules are downloaded from Kheyr servers and scored locally on your phone. High-score messages move to a Spam folder and do not trigger notifications.

Encrypted sync & desktop

Optionally sync SMS to the cloud with client-side encryption, then read and send messages from a native desktop app paired to your phone.

Getting started

Follow these steps after installing Kheyr from Google Play or a release build.

1. Install and open Kheyr

Launch the app and complete the welcome screen. Kheyr must become your default SMS app to read, send, and receive carrier SMS messages.

2. Grant required permissions

During onboarding, Kheyr requests permissions only for features you use:

  • SMS — read, receive, and send text messages as your default SMS app.
  • Contacts — show contact names and apply contact-specific notification rules.
  • Notifications — alert you for new messages after spam and privacy policies are applied.
  • Phone state — identify dual-SIM slots and route outgoing SMS to the selected SIM.
If you deny SMS permission or refuse the default SMS role, core messaging features remain unavailable until you grant them in Android settings.

3. Import existing messages

On first setup, Kheyr can import SMS already stored on your device into its local encrypted database. Deleted message history before sync is enabled is not uploaded to the cloud.

4. Download spam rules

Kheyr downloads the latest global spam rule set. Scoring runs entirely on your device; message bodies are not sent to the server for spam classification.

5. Start messaging

Use the floating compose button to start a conversation, reply in threads, retry failed sends, and choose a SIM on dual-SIM phones.

Using the inbox

Navigation drawer

  • All Messages — main inbox, newest activity first.
  • Spam — messages classified as spam; no notifications by default.
  • Archived — conversations removed from the main list.
  • Pinned — pinned threads sorted by pin date.
  • Contacts — quick access to people you message.
  • Desktop Sync — pair and manage desktop devices.
  • Settings — notifications, spam, SIM, sync, privacy, appearance.

Thread actions

Long-press a conversation to pin, archive, delete, mark as spam, mark as read, mute, or open notification settings for that thread.

Conversation screen

  • Send and receive SMS, including multipart long messages.
  • See sending, sent, delivered, and failed status; tap retry on failures.
  • Select SIM before sending on dual-SIM devices.
  • Search inside a conversation.
  • Emoji-only messages appear larger without a normal bubble.
  • Copy, delete, or correct spam classification on individual messages.

Spam protection

Kheyr scores each incoming SMS locally using server-managed global rules. The server provides rule definitions only; it does not need to read your message content to classify spam.

Default score behavior

ScoreClassificationBehavior
Below 40NormalAppears in All Messages; normal notifications apply.
40–69SuspiciousStays in inbox; details may show a warning.
70+SpamMoved to Spam folder; notification suppressed.

Rule types include number prefixes, keywords, URLs, regex patterns, sender frequency, OTP patterns with negative scores, and more. You can mark a thread or message as Spam or Not Spam; feedback is sent anonymously without uploading raw message bodies by default.

Notifications

Customize how Kheyr alerts you:

  • Global and per-thread ringtones.
  • Mute or silence specific conversations.
  • Separate policies for unknown senders: normal, silent, or no notification.
  • Notification content privacy: show preview, sender only, or hide content on the lock screen.
  • Spam and blocked senders never notify by default.

Dual SIM

Kheyr detects available SIM cards, shows which SIM received a message when the platform provides that data, and lets you:

  • Pick a SIM when composing or replying.
  • Set a global default SIM in Settings.
  • Override the default SIM per conversation.

Desktop send requests respect your thread default or global default SIM selection.

Optional sync and desktop access

Sync is off by default. You must explicitly create an account and enable sync in Settings.

When sync is enabled

  1. Sign in with your phone number and OTP verification.
  2. Kheyr uploads your current SMS messages using client-side encryption.
  3. Incremental sync keeps new messages, spam state, pins, archives, read state, and delete events in sync.
  4. Install the Kheyr desktop app on Windows, macOS, or Linux.
  5. Pair desktop by scanning the QR code on your phone and approving the device.
  6. Read synced threads on desktop and send SMS through your phone’s SIM.
SMS bodies are encrypted on your phone before upload. Kheyr servers store ciphertext and limited metadata only. See the Security page for encryption details.

Desktop SMS sending

Desktop does not send SMS directly. It creates a relay request; your Android phone must be online to deliver the message through the selected SIM. If the phone is offline, desktop shows a waiting state and retries according to queue policy.

Revoking access

View paired desktop devices under Settings → Desktop Devices and revoke any device instantly. You can also delete cloud data or your account from Privacy & Security settings.

Direct messages

When both you and a contact use Kheyr with accounts, you may send internet-based direct text and picture messages. These are visually distinct from carrier SMS. If the recipient is not registered, Kheyr falls back to SMS automatically.

Direct messages are transported over Kheyr’s online service. They are not end-to-end encrypted in the current release; see Security for the full encryption model.

Troubleshooting

  • Cannot send SMS — confirm Kheyr is the default SMS app and SMS permission is granted. Check SIM selection and carrier signal.
  • No notifications — verify notification permission, thread mute state, unknown-sender policy, and spam classification.
  • Sync not updating — ensure sync is enabled, you are signed in, and battery optimization is not restricting background sync.
  • Desktop send stuck — keep your phone online and signed in; relay requests expire if the phone stays offline too long.
  • Lost default SMS role — Kheyr warns you and prompts to restore the default SMS role in Android settings.
